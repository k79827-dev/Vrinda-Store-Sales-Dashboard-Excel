# Vrinda Store Sales Dashboard (Excel)

## Project Overview
This project is an interactive Excel dashboard built to analyze sales performance of Vrinda Store.

## Key Insights
- Women customers contribute ~65% of total sales
- Maharashtra, Karnataka, and Uttar Pradesh are top-performing states
- Age group 30-49 contributes the highest sales (~50%)
- Amazon, Flipkart, and Myntra are major sales channels

##  Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
